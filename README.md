## 🌟 Features

- ✅ Auto View Status
- ✅ Auto React to Status
- ✅ Anti-Call Protection
- ✅ Anti-Delete Detection
- ✅ View-Once Media Saver
- ✅ Stylish Menu with Images
- ✅ Uptime Monitor
- ✅ Owner Info Display
- ✅ WhatsApp Channel Integration

## 📱 Commands

| Command | Description |
|---------|-------------|
| `.menu` | Show main menu |
| `.ping` | Check bot response time |
| `.uptime` | Show bot uptime |
| `.owner` | Show owner information |
| `.channel` | Get channel link |
| `.autoviewstatus on/off` | Auto view status updates |
| `.autoreactstatus on/off` | Auto react to status |
| `.anticall on/off` | Block incoming calls |
| `.antidelete on/off` | Detect deleted messages |
| `.vv` | Save view-once media |

## 🚀 Deployment

### Heroku
1. Click the Deploy button above
2. Add your session ID
3. Set owner number
4. Deploy!

### Render
1. Fork this repository
2. Create new Web Service on Render
3. Connect your repository
4. Add environment variables
5. Deploy!

## ⚙️ Environment Variables

| Variable | Description |
|----------|-------------|
| `SESSION_ID` | Your WhatsApp session ID |
| `OWNER_NUMBER` | Owner's phone number |
| `BOT_NAME` | Bot display name |
| `PREFIX` | Command prefix (default: .) |
| `CHANNEL_LINK` | WhatsApp channel URL |
