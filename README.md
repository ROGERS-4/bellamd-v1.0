# 📋 COMMANDS LIST

|--------|------------|--------|
| `.menu` | Show animated menu | `.menu` |
| `.ping` | Check bot status | `.ping` |
| `.uptime` | Show uptime | `.uptime` |
| `.owner` | Owner info | `.owner` |
| `.channel` | Channel link | `.channel` |
| `.autoviewstatus` | Auto view status | `.autoviewstatus on/off` |
